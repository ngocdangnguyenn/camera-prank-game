# Camera Prank Game - Cybersecurity Education Project

## Overview
Educational project for Vietnamese high school students to raise awareness about online scams and phishing attacks through interactive demonstration.

## Purpose
- Teach students to recognize online deception techniques
- Develop critical thinking when encountering suspicious websites
- Raise cybersecurity awareness through hands-on experience
- Understand psychological manipulation in digital environments

## How It Works
1. **Start**: Students scan QR code and enter their name
2. **Camera Request**: Website asks for camera permission for "profile photo"
3. **Photo Capture**: Students take a photo thinking it's for a game
4. **Loading Screen**: Shows fake profile creation with their photo
5. **The Reveal**: Truth revealed - no game exists, data was collected
6. **Education**: Explains risks and safety tips

## Educational Value
Demonstrates how:
- Fake websites can appear legitimate and trustworthy
- Personal information can be collected under false pretenses
- Images and names can be misused for identity theft
- Importance of verifying sources before sharing personal data

## Technology
- Pure HTML, CSS, JavaScript
- Camera API for photo capture
- No external dependencies
- Responsive design for mobile devices

## Usage Guide

**For Teachers:**
1. Deploy to GitHub Pages
2. Generate QR code from URL
3. Present as "fun game" to students
4. Use results for cybersecurity discussion

**For Students:**
1. Scan QR code
2. Enter name and take photo
3. Experience the "prank"
4. Learn about online safety!

## Deployment
1. Push to GitHub repository
2. Enable GitHub Pages in Settings
3. Generate QR code from GitHub Pages URL
4. Ready to use in classroom
